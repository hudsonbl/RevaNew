How to use: while entering dates please enter in yyyymmdd format. If you want to refernce a specific account in any of the features, reference it exactly by name. If you type name of target account incorrectly it doesn't save newly added item. Some of the features still need to be updated. In expenses, if you want to Items to be in the same category, you also have to type the name to be exact with recent entries. Otherwise in Statistics, it will treat it as a new category.
